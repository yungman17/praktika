# Рауф — Система управления библиотекой

Итоговый проект по лекциям 6.1 и 6.2 (C# Windows Forms, .NET 8).
Простое приложение с тёмной темой.

## Возможности (по требованиям из лекции 6.1)
- **Книги** — добавление, редактирование, удаление, поиск по названию/автору/ISBN, статус (в наличии / выдана)
- **Читатели** — регистрация, телефон, дата регистрации, удаление
- **Выдача книг** — выдача на 14 дней, возврат, расчёт штрафа за просрочку (10 руб/день)
- **Отчёты** — статистика, список должников со штрафами, популярные книги
- **Сохранение в JSON** — файл `library.json` создаётся рядом с программой автоматически
- **Валидация и обработка ошибок** — нельзя удалить выданную книгу или читателя-должника

## Как запустить
1. Установить .NET 8 SDK (если нет): https://dotnet.microsoft.com/download
2. В этой папке выполнить:
   ```
   dotnet run
   ```
   Или открыть папку в Visual Studio 2022 и нажать F5.
