Автор: **Рауф**

## Введение
Программа предназначена для автоматизации учёта книг, читателей и выдачи литературы. Данные сохраняются автоматически в файл `library.json` рядом с программой.

## Начало работы
1. Запустите приложение (`dotnet run` в папке LibrarySystem или F5 в Visual Studio).
2. Откроется главное окно с четырьмя вкладками: **Книги**, **Читатели**, **Выдача книг**, **Отчёты**.
3. Внизу окна отображается статистика: количество книг, читателей и активных выдач.

## Работа с книгами
**Добавление:** вкладка «Книги» → кнопка «➕ Добавить» → последовательно введите название (обязательно), автора (обязательно), жанр, год, ISBN.

**Поиск:** введите текст в поле поиска — фильтрация идёт сразу по названию, автору и ISBN.

**Редактирование:** выберите книгу в таблице → «✏ Изменить» → поля откроются с текущими значениями.

**Удаление:** выберите книгу → «🗑 Удалить» → подтвердите. Выданную читателю книгу удалить нельзя — программа сообщит об ошибке.

## Работа с читателями
**Регистрация:** вкладка «Читатели» → «➕ Добавить» → введите ФИО (обязательно) и телефон. Дата регистрации ставится автоматически.

**Удаление:** выберите читателя → «🗑 Удалить». Читателя с невозвращёнными книгами удалить нельзя.

## Выдача и возврат книг
**Выдача:** вкладка «Выдача книг» → «📖 Выдать книгу» → выберите книгу из списка доступных → выберите читателя. Срок выдачи — 14 дней, книга получает статус «Выдана».

**Возврат:** выберите запись в таблице → «✅ Вернуть книгу». Если срок просрочен, программа покажет сумму штрафа (10 руб. за каждый день просрочки).

**Статусы в таблице:** «На руках», «ПРОСРОЧКА!», «Возвращена».

## Отчёты
Вкладка «Отчёты» → «📋 Сформировать отчёт». Отчёт включает:
- общую статистику (всего книг, в наличии, выдано, читателей);
- список должников с суммами штрафов;
- топ-3 популярных книги по числу выдач.

## Запуск тестов
В корневой папке выполните `dotnet test` — запустятся юнит-тесты контроллеров (16 тестов: книги, читатели, выдача, штрафы).

## Возможные проблемы
| Проблема | Решение |
|---|---|
| «Нельзя удалить книгу, которая выдана» | Сначала оформите возврат на вкладке «Выдача книг» |
| «У читателя есть невозвращённые книги» | Верните все книги читателя, затем удаляйте |
| Данные пропали | Проверьте, не удалён ли файл `library.json` рядом с .exe |
