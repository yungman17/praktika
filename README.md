# Рауф — Система управления библиотекой

## Как открыть
Двойной клик по **Рауф.sln** → откроется в Visual Studio 2022 Community с двумя проектами.
Запуск приложения — клавиша **F5**. Тесты — меню **Test → Test Explorer**.

Нужна нагрузка «.NET desktop development» в Visual Studio Installer (там же ставится .NET 10).

## Структура
- `Рауф/` — приложение (Models.cs, Controllers.cs, DataService.cs, MainForm.cs, Program.cs)
- `Рауф.Tests/` — юнит-тесты контроллеров (16 шт.)
